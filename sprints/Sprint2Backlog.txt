User Stories from Product Backlog that we will address in this sprint:
a) As a user, I want to print the whole path of the current	working directory. [Pwd]
b) As a user, I want to create directories relative to current directory, or create a full path of directories. [Mkdir]
c) As a user, I want to print contents (a file or a directory) of the current directory, with a new line following each Content.
   I also want to print the file if give a path to a file or print the contents of a directory if I give a path to a directory.[Ls]
d) As a user, I want to change my directory to another directory that may be relevant to the current directory, or to a different full path. [Cd]
e) As a user, I can view my user command history from most recent one to the oldest. I can also choose to display the history from a given number[History[Number]]
f) As a user, I want to print documentation for CMD. [MAN CMD]
g) As a user, I want to see the contents of the one or more files in the shell, separated by three line breaks if there is more than one file.  [CatFILE1[FILE2]]   
h) As a user, I want to be able to save the current working directory and keep track of the previous directories. [Pushd]
i) As a user, I want to return to the previous working directory. [PopDIR]
j) As a user, I want check whether my command is valid or not.
k) As a user, I want to be able to type the commands into the shell. [JShell]

User Stories from Product Backlog that we will address in this sprint:
a) As a user, I want to print the whole path of the current	working directory. [Pwd]
b) As a user, I want to create directories relative to current directory, or create a full path of directories. [Mkdir]
c) As a user, I want to print contents (a file or a directory) of the current directory, with a new line following each Content. [Ls]
d) As a user, I want to print a file if I give a path to a file, or print the contents of a directory, if I give a path to a directory.[Ls[Path]]
e) As a user, I want to change my directory to another directory that may be relevant to the current directory, or to a different full path. [Cd]
f) As a user, I want check whether my command is valid or not.
g) As a user, I can view my user command history from most recent one to the oldest. I can also choose to display the history from a given number [History[Number]]
h) As a user, I want to print documentation for CMD. [MAN CMD]
i) As a user, I want to be able to type the commands into the shell. [JShell]

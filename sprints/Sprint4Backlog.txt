November 12 to November 18


User Stories from Product Backlog that we will address in this sprint:
a) As a user, I want to see the contents of more than one file in the shell,
      separated by three line breaks if there is more than one file. [CAT]
b) As a user, I want to be able to redirect output of commands to a file [>, >>]
c) As a user, I want to be able to recursively list all sub directories. [Ls]
d) As a user, I want to be able to find files or directories with a specific
      name by giving one or more directory paths. [Find]
e) As a user, I want to be able to move an item from one directory to another. [Mv]
f) As a user, I want to be able to copy an item from one directory to another. [Cp]
g) As a user, I want to be able to retrieve a file by giving a URL.
      A file should be created in the current working directory.  [Get]
